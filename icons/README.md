# CruiseCrew PWA icons

Place the final CruiseCrew PNG logo assets in this folder so they are served from the deployed site root, for example `https://getcruisecrew.com/icons/icon-192.png`.

Required files:

- `icon-192.png` — 192×192 PNG app icon used by the web app manifest.
- `icon-512.png` — 512×512 PNG app icon used by the web app manifest.
- `icon-maskable-512.png` — 512×512 PNG maskable icon with safe-zone padding for Android launchers.
- `apple-touch-icon.png` — PNG icon used by iPhone/iPad when adding CruiseCrew to the Home Screen.

These placeholder paths are referenced by `/manifest.webmanifest` and `index.html`; replace/add the PNG files above before final production branding verification.
